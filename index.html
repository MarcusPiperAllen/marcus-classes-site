Marcus's Favorite NOVA Classes

ITE 152 - Introduction to Information and Technology

This class is a great introduction to computers and digital literacy. It's ideal for beginners who want to understand how computers work and learn what it means to be a digital citizen.

ITD 110 - Introduction to Web Development

This class is a great starting point for learning how to build websites from scratch. Although it can be challenging online, it strengthens your coding journey and creativity.

ITN 170 - Networking Concepts

This course explains the core concepts of networking and topologies. Labs were challenging, but I learned valuable command-line tools — I just need more chances to apply them in the real world.

ITN 170 - Linux System Administration

This class introduces the basics of Linux. It’s perfect for anyone interested in using Linux for IT work. I hope to apply this knowledge on the job one day.
